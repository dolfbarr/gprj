# Installation

```sh
$ npm install gprj -g
```

# Usage
<!-- usage -->
```sh-session
$ npm install -g gprj
$ gprj COMMAND
running command...
$ gprj (--version|-v)
gprj/1.0.0 darwin-x64 node-v14.17.3
$ gprj --help [COMMAND]
USAGE
  $ gprj COMMAND
...
```
